# SV2 Surface Assistant

The assistant is not an unrestricted tutor or live LLM. It is a bounded explanation layer for this artifact. It can explain lessons, provide teacher-note style answer guidance, describe printing/autosave/teacher mode, and disclose source limits.

It should not claim private classroom context, school policy, grades, medical advice, or unsafe experiment guidance.
