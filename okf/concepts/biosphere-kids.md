# BioSphere Kids

BioSphere Kids is a static browser lesson surface. It renders printable, mobile-friendly lessons with a fixed five-section structure: Learn, Observe, Think, Connect, and Review.

This imported deployment includes Science, Geography, and History lessons. The assistant answers from the imported lesson JSON and local project metadata.
