# OKF Layer

The OKF-style layer lets external agents inspect the whole artifact instead of only the currently rendered lesson. It is behind the scenes for humans, but public and readable for agents.

The OKF layer does not replace the lesson renderer. It documents the source, boundaries, build contract, and validation path.
