# Assistant Intents

The assistant should route these families before generic fallback:

- identity and orientation
- lesson inventory
- subject overview
- lesson summary
- teacher-note answer guidance
- print mode
- teacher mode
- autosave/localStorage
- source/provenance
- OKF/agent-readable inspection
- unsafe/medical boundary
- private classroom data boundary
- off-topic boundary

It should be able to answer about all 12 lessons, not only the currently rendered lesson.
