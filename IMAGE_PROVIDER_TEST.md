1. **Primary Source:** Use Pexels images as primary choice

   - Better attribution for educational context
   - More consistent with premium lesson design

2. **Secondary Source:** Use Pixabay when Pexels lacks coverage
   - Good fallback for specific search terms
   - Still CC0 licensed and free to use
   - Manually verify image quality before using

3. **Best Practice:** Maintain both APIs in rotation
   - Provides redundancy if one API has downtime
   - Allows flexible sourcing based on search results
   - Easy to switch if one provider improves/declines
